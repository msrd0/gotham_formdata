Trait gotham::state::StateData

pub trait StateData: Any + Send { }
Expand description

A marker trait for types that can be stored in State.

This is typically implemented using #[derive(StateData)].

use gotham::state::StateData;

#[derive(StateData)]
struct MyStateData {
    x: u32,
}
