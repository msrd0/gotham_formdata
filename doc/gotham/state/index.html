Defines types for passing request state through Middleware and Handler implementations

Structs

State
Provides storage for request state, and stores one item of each type. The types used for storage must implement the StateData trait to allow its storage, which is usually done by adding #[derive(StateData)] on the type in question.

Traits

FromState
A trait for accessing data that is stored in State.
StateData
A marker trait for types that can be stored in State.
